# Mandatory Hands-on 1 - Spring Data JPA

## Objective
Retrieve all countries from the MySQL database using Spring Data JPA.

## Technologies Used
- Java 21
- Spring Boot 3.5.16
- Spring Data JPA
- Hibernate
- MySQL 5.7
- Maven

## Project Structure
- Entity: Country
- Repository: CountryRepository
- Service: CountryService
- Main Class: OrmLearnApplication
