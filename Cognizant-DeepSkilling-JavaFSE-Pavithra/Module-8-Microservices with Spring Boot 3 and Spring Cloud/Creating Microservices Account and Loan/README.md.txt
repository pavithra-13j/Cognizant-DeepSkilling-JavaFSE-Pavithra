# Creating Microservices for Account and Loan

## Objective
This project demonstrates the implementation of two independent Spring Boot microservices:
- Account Microservice
- Loan Microservice

## Technologies Used
- Java
- Spring Boot
- Maven

## Project Structure
- account/
- loan/
- screenshots/

## REST Endpoints

### Account Microservice
GET /accounts/{number}

Example:
http://localhost:8080/accounts/12345

### Loan Microservice
GET /loans/{number}

Example:
http://localhost:8081/loans/100
